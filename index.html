About "Clean Air" Project

A weather application that interacts with the MapBox AccuWeather and Weather stack. The project was originally designed for cell phones and then adopted for a large screen size. The application shows current weather conditions and predicts the forecast for the four subsequent days. The air quality index (AQI) is also included. For this project, only HTML, CSS and Javascript were used in the front end. The application has its front end in the browser where users provide their location. Behind the scenes, using Node.js, the application communicates with third party servers to convert that location into the forecast and the AQI. In this application, asynchronous Node.js was used.

About "Purr & Woof" Project

The "Purr and Woof" project was made completely from scratch. The initial ideas were planned on paper, and then transferred to the interface design application, Figma (the project UI/UX prototype can be seen here). Express, MongoDB, and a variety of npm modules, such as multer, bcryptjs, gravatar, etc. were used in the back end of the application. As a result, an extensive back end API was built, as well as protected routes/endpoints with JSON Web Tokens. The front end was built with React JS, along with Redux for app state management. A widely-used CSS framework (Bootstrap4) was used in the app as well.
